## New

- **Contact page** — send messages directly via a contact form. Emails are delivered through Resend with a branded template matching the rest of the app.
- **About page** — an overview of the project, its creator, and the Darrington Press Community Gaming License.
- **Privacy Policy** — full disclosure of data collection, third-party services, and user rights.
- **Outdated version banner** — users on an old cached version of the app are prompted to refresh.
- **Theme toggle on public pages** — light/dark/system mode is now accessible from the header on all public-facing pages.

## Redesigned

- **Landing page** — fully redesigned with a cleaner layout and clearer call-to-action sections.
- **Reference pages** — ancestries, classes, beastforms, environments, and domains pages refreshed with consistent headers and improved filter controls.
- **Homebrew page** — personal post list and card layout redesigned for better readability.
- **Community empty states** — empty community pages now show a more polished illustrated placeholder.
- **Collapsible section trigger** — refactored for consistency across the card and adversary creators.

## Improvements

- **Shared page header component** — all dashboard pages now use a unified `PageHeader` component, and subtitle copy has been rewritten throughout.
- **Standardised filter controls** — reference and community pages use a consistent `MultipleSelector` pattern with grid layouts that don't shift when selections change.
- **Footer** — condensed to two columns; all site pages including Contact are linked under a single Pages list.
- **Adversary creator layout** — switched from flex to CSS grid so the preview column no longer changes width when collapsible forms expand or collapse.
- **Dice roller shapes** — the damage tab buttons now use polygon shapes that match each die type (d4 triangle, d6 square, d8 diamond, and so on).
- **Sidebar divider** — visual separator added between navigation sections in the dashboard sidebar.

## Bug Fixes

- Input, textarea, and select backgrounds now render white in light mode instead of inheriting the tinted page background.
- Preview modals were incorrectly oversized on some screen sizes.
- Fixed a table hydration error in server-rendered pages.
- Beastform tier filter no longer causes the search bar to resize when multiple tiers are selected.

## Internal

- shadcn/ui updated to the latest release.
- Version instrumentation added for cache-busting.
- Seed data updated — local development now includes test users and sample homebrew content loaded automatically via Docker.
